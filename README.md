# Shawn Riju | Digital Portfolio

An interactive and optimized portfolio showcasing my professional journey, software projects, technical expertise and personal story.

## Live Demo
**URL**: [https://shawnriju.vercel.app](https://shawnriju.vercel.app)

## Tech Stack
- **Frontend**: React (Vite), TypeScript
- **Styling**: Tailwind CSS, shadcn/ui
- **Animations**: Framer Motion
- **Hosting**: Vercel

## Interactive Features
- **Project Case Studies**: Detailed breakdowns of software and infrastructure work.
- **Protocol Shenron**: A custom-built scavenger hunt integrated into the UI.
- **Dynamic Interface**: A Nier-inspired aesthetic with scroll-synchronized progress tracking.
